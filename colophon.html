Colophon · about this site

Built with intention.

A short note on what this site is made of, why each choice was made, and what's left to ship.

Type

Display
Cormorant Garamond — Christian Thalmann, OFL. Drawn after Claude Garamond's 16th-century romans; ideal for the editorial register the site is trying to occupy.
Body
Times New Roman — Monotype, 1932, system fallback. Chosen against a sans-serif. Body and display in the same idiom keeps the page reading as a single document, not a UI.
Labels & code
JetBrains Mono — Jet Brains s.r.o., OFL. The third role: small-caps tracking, code blocks, terminal-flavoured surfaces.

Palette

Two-palette system, user-toggleable. Default — Brass on Black: off-black ground (#0A0A0C), parchment cream ink (#F0EAD6), antique brass accent (#B6884A). Maps to alternative-asset houses — Apollo, Blackstone, KKR. Alt — FT Pink: cream-pink (#FFF1E5) Financial Times newsprint colour since 1893, ink-navy (#0E1A2B), burgundy (#990F3D). Light editorial alternative for daytime reading.

Text contrast lifted across both palettes to address eye-strain on extended reading. Minimum 4.5∶1 for body text on both grounds.

Stack

HTML
Hand-written. Main page + tearsheet + colophon + privacy notice + legal notice.
CSS
~2,600 lines, no framework. CSS custom properties drive the palette swap.
JS
Vanilla. Project panel, CV modal, IntersectionObserver-driven nav state, Chart.js for the strategy equity curve.
Charts
Chart.js 4.4.4, loaded with SRI hash from jsDelivr.
Hosting
Portfolio: Netlify (static, auto-deploys from GitHub). Finance Analyser: Render (Python/FastAPI, free tier). No analytics, no cookies, no trackers.

Inspiration

Financial Times print edition · Bloomberg Terminal · Apollo Global annual report · the Jane Street recruiting site · Robin Rendle's writing site · the typography of The Economist's Espresso app.

What's not here yet

A monthly research letter. A French-language version. A live equity-curve sparkline in the hero pulling from the Alpaca paper account. A /now page. Eventually, an Equity Research Initiation Note as the first formal long-form publication. Custom domain (finnlakin.com) once purchased.
